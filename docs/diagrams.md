## 📊 Diagramas

O projeto está documentado através dos seguintes diagramas:

### 📌 Casos de Uso

Representam as principais interações do usuário com o sistema:

- Autenticação
- Funcionalidades da aplicação
  - Projetos
  - Status (por projeto)
  - Tarefas
  - Configurações

👉 Acesse: [Casos de Uso](./diagrams/use-cases.md)

---

### 🔄 Sequência

Representam o fluxo de execução das principais operações:

- Login (Email e Senha)
- Cadastro (Sign Up)
- Login/Cadastro com Google/GitHub
- CRUD de Projetos
- CRUD de Status
- CRUD de Tarefas

👉 Acesse: [Sequência](./diagrams/sequences.md)

---

### 🧠 Estados

Representam os estados possíveis das entidades do sistema:

- Autenticação
- Projetos
- Status do Projeto
- Tarefas

👉 Acesse: [Estados](./diagrams/states.md)

---

### 🧩 Entidade-Relacionamento (ER)

Representa a estrutura de dados e relacionamentos do sistema:

- Usuários e Projetos
- Tarefas e Membros

👉 Acesse: [ER](./diagrams/er.md)
